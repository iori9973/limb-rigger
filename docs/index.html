Limb Rigger

VRChat アバター用のサブアーム・義手・義足セットアップ自動化 Unity Editor 拡張

VCC に追加する

以下のボタンをクリックすると VCC が起動し、Limb Rigger のリポジトリが追加されます。

Add to VCC

ボタンが反応しない場合は、VCC の Settings → Packages → Add Repository に以下の URL を貼り付けてください:

https://iori9973.github.io/limb-rigger/index.json

追加後のインストール手順

1

VCC で対象プロジェクトの Manage Project を開く

2

パッケージ一覧から Limb Rigger を探して Install

3

Unity で Tools → Limb Rigger メニューからウィンドウを開く
